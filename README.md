# 🏗️ Auto-RAG Architect

In the world of **AI Strategy**, information retrieval is the backbone of accuracy. Generic RAG implementations often fail because they don't account for specific data structures. **Auto-RAG Architect** is a technical framework that analyzes your data and instantiates the most effective retrieval pipeline automatically.

## 🚀 Strategic Vision
Building a RAG system shouldn't be a guessing game. This tool enables organizations to:
- **Minimize Hallucinations**: By choosing the right retrieval strategy (Semantic vs Keyword vs Hybrid).
- **Optimize Performance**: Automated chunking logic tailored to document types.
- **Scale Intelligence**: A plug-and-play architecture that connects to multiple vector stores.

## 🛠️ Intelligent Features
- **Strategy Selector**: Automatically switches between `Dense`, `Sparse`, and `Hybrid` retrieval based on query intent.
- **Chunking Optimizer**: Dynamic sliding window chunking to preserve semantic context.
- **Vector Store Abstraction**: Support for ChromaDB, Pinecone, and Weaviate logic.
- **Evaluation Loop**: Built-in logic to measure retrieval precision.

## 🏗️ Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Run the architect: `python main.py --data ./path/to/docs`
